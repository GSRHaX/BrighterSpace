# BrighterSpace 🤖

### [ Client-side RAG system for BrightSpace ] 😵‍💫

Core Features:

 - Semantic search + AI support. Can search up any activity from any class, any event from any calendar, and get detailed info from AI.
 - Ambient AI integration. AI helps you out without asking. (Be it reminders, suggestions, summarization, grade monitoring);
 - Smart note, bookmark, link keeping. Dump notes, links, or bookmark speicifc activities to dashboard. Have them accesible at the click of a button. (dynamic based on page, actively suggesting based on semantic search)


UI Features:

 - Custom banner backgrounds for dashboard and classes
 - Declutter Mode, can remove widgets per class / dashboard. Get rid of brightspace / teacher junk
